# J_Tracker_Checkboxes v2.0.0

## Nuke Tracker4 tool that bulk-toggles the Translate / Rotate / Scale checkboxes for:
	•	All tracks, or
	•	Only the currently selected tracks

## Where it lives
	•	Inside the Tracker4 node
	•	Under the Check Boxes tab

## How to use
	1.	Create or select a Tracker4 node with multiple tracks
	2.	Optionally: in the Tracker UI, select some tracks (for “selected tracks” mode)
	3.	Open the Check Boxes tab
	4.	Apply to all tracks or selected tracks
	5.	Toggle translate / rotate / scale on or off
	6.	Click execute

## Install
	•	Put J_Tracker_Checkboxes.py in your tools folder (for example, in ~/.nuke/python/)
	•	In menu.py write import J_Tracker_Checkboxes
	•	Restart Nuke and create a new Tracker4 node
